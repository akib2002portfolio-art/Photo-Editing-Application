📸 Photo Editor
A responsive web-based photo editing application that enables users to enhance and customize images directly in the browser.

Features
Upload and preview images
Adjust brightness, contrast, saturation, and blur
Rotate and flip images
Crop images
Reset edits instantly
Save edited images to device
Responsive user interface
Technologies Used
HTML5
CSS3
JavaScript
Cropper.js
Live Demo
🔗 https://swapan-code.github.io/Photo-Editor-Project/

Learning Outcomes
Image processing in the browser
JavaScript DOM manipulation
Canvas and image editing techniques
Responsive UI design
